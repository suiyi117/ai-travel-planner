## 工作台怎么用

1. 在 AeroTravel 完成路线、偏好并生成/编辑行程。
2. 打开顶部 **导出** 菜单：
   - **预览专属页**：浏览器打开只读客户页预览。
   - **发布专属行程**：下载自包含 `trip-{token}.html` + JSON，并复制发布说明与客户文案。
   - **导出总览图**：竖向 3:4 风格总览 PNG（私信/小红书）。
   - **导出 PDF 备份**：打开打印页，另存为 PDF。
   - **复制客户行程 / 导出客户长图 / 导出日历**：沿用原有交付能力。

## 人工发布（第一阶段）

1. 将 `trip-{token}.html` 上传到静态托管。
2. 建议 URL：`https://你的域名/t/{token}.html` 或 `/t/{token}/index.html`。
3. 页面已带 `noindex`，token 为高强度随机串；勿写入证件号、完整订单号、手机号。
4. 向客户同时发送：专属链接 + 可复制文字 + PDF 备份。
5. 需要改行程时：在工作台改结构化行程后重新发布（覆盖同路径或更换新 token）。

## 技术边界

- 不引入账号、数据库、在线支付。
- 高德跳转使用官方 HTTPS URI；不承诺多日路线一键导入。
- 总览地图编号与每日锚点来自同一份 `trip package`。
- 估算连线在客户页用虚线表示，不伪装成真实道路。
